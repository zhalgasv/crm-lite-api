# CRM Lite API

CRM Lite API is a simple backend application for managing customers in a CRM system.

## Tech Stack

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Flyway
- Maven
- JUnit 5
- MockMvc
- Docker Compose

## Features

- Create customer
- Get all customers
- Get customer by id
- Update customer
- Delete customer
- Validation
- Global exception handling
- Repository tests
- Controller tests

docker compose up -d
